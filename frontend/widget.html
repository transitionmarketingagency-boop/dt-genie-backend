Embeddable AI Chatbot Widget

This widget can be embedded on any website. Copy the script below and paste it before the closing </body> tag.

How to Use This Widget

  1. Make sure your backend server is running on http://localhost:5000
  2. Copy the entire <script> section above (starting from <div id="dtm-chatbot-widget">)
  3. Paste it before the closing </body> tag on your website
  4. The chat widget will appear in the bottom-right corner

For production: Change BACKEND_URL to your deployed backend URL (e.g., your Replit URL)
